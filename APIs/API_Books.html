BookStore Implementation - BookSearch

Application Programming Interface

Database version

    getAllBooks()
  • Displays all books found in the BookStore book table.
    getBookByIsbn(isbn)
  • Returns a unique example of a book (as a book object) identified by the supplied isbn, if found; the object will be empty if not found.
  • isbn - a string containing the isbn to search for.
    getBookByTitle(title)
  • Returns a unique example of the book (as a book object) with the supplied title, if found; the object will be empty if not found.
  • title - a string containing the book title to search for.
    getBooksByAuthor(author)
  • Returns an array of book objects if any are found; an empty array if not.
  • author - a string containing the name of the author to search for.
    getBooksByPublisher(publisher)
  • Returns an array of book objects if any are found; an empty array if not.
  • publisher - a string containing the name of the publisher to search for.
    getBooksUsingKeywords(words, searchParam)
  • Returns an array of book objects if any are found; an empty array of not.
  • words - a comma separated string of words to search for filtered by the searchParam.
  • searchParam - title is the only parameter by which you can search using key words in this API.
    showBooks(searchResults)
  • Displays the books found in previous searches.
  • searchResults - an array of book objects if any are found; an empty array if not.
    getAuthors()
  • Returns an array of unique author names in ascending order by whole name.
    getPublishers()
  • Returns an array of unique publisher names in ascending order.

Housekeeping

    deleteAllBooks()
  • Drops the book table and recreates it without any books.
    reloadBooks()
  • Drops the book table and recreates it before reloading it with the original book data data.

Utilities

    convertToObject(string)
  • Converts data from the database into objects or object collections. You do not need ever to call this function that is interal to the API.
    convertToArray(string)
  • Converts data from the database into an array. You need never call this function that is internal to the API.
